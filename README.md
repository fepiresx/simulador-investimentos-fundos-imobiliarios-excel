## 💜 About the Project

This project was developed as part of the **Santander - Excel with Artificial Intelligence Bootcamp** on the DIO platform.

The goal was to create a **financial investment simulator** focused on real estate funds (FIIs), enabling users to explore different scenarios and make **strategic, data-driven decisions**.

This solution was built under the concept of **NYKOS CAPITAL**, a fictional brand that represents **discipline, strategy, and long-term wealth building**.

## ▶️ How to Use

1. Download the Excel file
2. Open it in Microsoft Excel
3. Enter your investment parameters:

   * Initial investment
   * Monthly contribution
   * Investment period
   * Expected return rate
4. Analyze projections and financial outcomes
